accountAccessRoleName: string

AWS Account Access Role Name

Description

The name of the role that will be assumed to accept invitation from the account invited to AWS Organizations. This rule must be present in the account invited to AWS Organizations and Management account must be able to assume the role.

email: string

Optional Readonlytags

tags?: Tag[]

A list of tags that you want to attach to the account when it becomes a member of the organization. For each tag in the list, you must specify both a tag key and a value. You can set the value to an empty string, but you can't set it to null.
