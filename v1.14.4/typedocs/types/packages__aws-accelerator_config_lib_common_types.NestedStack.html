    Nested Stack Type

    Type definition for nested CloudFormation stacks within ASEA deployments. Represents child stacks that are created as resources within parent stacks, maintaining the hierarchical relationship and complete resource inventory.

    Key Features

    • Hierarchical Structure: Maintains parent-child stack relationships
    • Complete Inventory: Full resource tracking for nested stacks
    • Identity Management: Unique identification within parent context
    • Migration Support: Enables nested stack migration to LZA
