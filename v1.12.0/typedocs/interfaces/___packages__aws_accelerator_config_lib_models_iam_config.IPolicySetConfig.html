identityCenterDependency?: boolean

Flag indicates if the policy is used in Identity Center PermissionSet assignments.

Remarks

When the policy is used in Identity Center PermissionSet assignments, policy must be present in each deployment target accounts of identityCenterAssignments. When this flag is set to true policy is created in dependency stack.
